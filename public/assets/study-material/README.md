# Study Material PDF

This directory should contain a PDF file named `road-safety-study-material.pdf`.

## File Location
The PDF should be placed at: `public/assets/study-material/road-safety-study-material.pdf`

## Content Suggestions
The PDF should include:
- Road Safety Rules and Regulations
- Traffic Signs and Signals
- Helmet and Seatbelt Guidelines
- Speed Limits and Safety Measures
- Pedestrian Safety Guidelines
- Emergency Procedures
- Best Practices for Event Organisers

## File Size
Recommended file size: Under 5MB for easy download.

## Format
Standard PDF format (PDF/A recommended for accessibility).
